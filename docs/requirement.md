# 用户需求

从用户的视角来看，整个流程如下：

1. 上传与处理
   打开应用后上传视频，并填写视频名称，方便后续辨认。点击确定后等待视频处理，后端会异步进行处理，完成后再通知前端。

2. 选择检索范围
   视频处理完后，用户可以选择要检索的视频。检索范围不限于本次上传的内容，也可以包含之前已经处理好的视频，支持一起进行检索。

3. 执行检索
   选择完视频后，设定时间区间，并输入要检索的特征（例如"穿着白色衣服的人"这类自然语言）。点击搜索后，系统会返回所有匹配的画面(图片)。

# 实现状态

✅ **已完成实现**

## 技术栈
- 后端: FastAPI
- 目标检测: YOLOv8 (yolov8n)
- 衣物识别: OpenAI CLIP (clip-vit-base-patch32)
- 数据库: SQLite
- 前端: 简单 HTML/JS (单页应用)

## 文件结构
```
main.py              - FastAPI 应用 (包含前端页面)
db.py                - 数据库初始化
video_processor.py   - 视频处理与 YOLO 检测 + CLIP 衣物识别
search.py            - 搜索功能 (支持衣物特征检索)
docs/design.md       - 设计文档
data/
├── videos/          - 上传的视频
└── frames/          - 提取的帧图片
skyeye.db            - SQLite 数据库
```

## 运行方式
```bash
python main.py
# 访问 http://localhost:8000
```

## API 端点
- `GET /` - 前端页面
- `GET /api/videos` - 获取视频列表
- `POST /api/videos/upload` - 上传视频
- `POST /api/search` - 搜索
- `GET /api/frames/{frame_id}` - 获取帧图片
- `GET /api/frames/{frame_id}/annotated` - 获取带框的图片

## 新增功能: 衣物识别

### CLIP 衣物检测
- 使用 OpenAI CLIP 模型进行零样本衣物分类
- 支持检测的衣物类别:
  - 上装: 衬衫、T恤、毛衣、外套、大衣、夹克、运动衫
  - 下装: 裤子、牛仔裤、短裤、裙子
  - 颜色: 白色、黑色、红色、蓝色、绿色、黄色、深色

### 搜索语法
支持以下搜索方式:
- 基本 YOLO 类别: `person`, `car`, `dog`, `cat`
- 中文类别: `人`, `车`, `狗`, `猫`
- 衣物搜索:
  - 颜色 + 衣物: `白色衬衫`, `蓝色裤子`, `红色T恤`
  - 单独衣物: `牛仔裤`, `外套`, `毛衣`

### 返回结果
- 显示带绿色边框的匹配人物
- 显示衣物识别结果 (置信度百分比)
